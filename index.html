LERD is a holistic software development environment. Instead of describing algorithm via language LERD offers a powerful CAD WYSWYG design tool. Stop wasting your power to loading and running cryptic sources in your brain. Edit your program and instantly see how it works.

Key Features

Reactive & Interactive programming

LERD designed with Reactive programing paradigm which means graph. Main editor window shows graph as pipe and edit it as a generic text editor. Text and functions calls are shown in the same window with live execution arguments and results.

Layers

Conditions are implemented as layers. Each layer is linked to a pipe which determinate if the layer changes is applied. Also layer can be shown/hided via editor

Dependency request

Pipe holds information how pipes linked to each other. If pipe need an input it asks itself, if nothing found it asks parent pipe, if parent pipe has nothing it ask another parent and so on.
